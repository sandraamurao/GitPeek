# git.peek

A GitHub profile search app built with React. Search any GitHub username to instantly view their profile, stats, and top repositories.
> Built to practice React and working with third-party REST APIs.

🔗 [Live Demo](https://sandraamurao.github.io/git-peek) | 🔗 [GitHub Repo](https://github.com/sandraamurao/GitPeek)

---

## Features

- Search any GitHub user by username
- View profile, bio, location, and social stats
- Top repositories sorted by stars with language, forks, and watchers
- Responsive design with mobile support
- Error handling for invalid usernames

---

## Built With

- React
- Tailwind CSS
- GitHub REST API
- Vite
